Scheduling Algorithms

Learn about different CPU scheduling algorithms

FCFS

Type: Non-Preemptive

Processes are executed in the order they arrive. Simple and fair but can cause long waiting times.

SJF

Type: Non-Preemptive

Executes the process with the shortest burst time. Minimizes average waiting time.

Non-Preemptive Priority

Type: Non-Preemptive

Processes with higher priority are executed first. Can lead to starvation.

Round Robin

Type: Preemptive

Each process gets a time quantum. Fair distribution but may increase context switches.

SRTF

Type: Preemptive

Preempts when shorter process arrives. Optimal for average waiting time.

Preemptive Priority

Type: Preemptive

Higher priority process preempts lower priority. Responsive to important tasks.

Priority with Round Robin

Type: Preemptive

Combines priority and round robin. Processes ready at same priority use RR.
